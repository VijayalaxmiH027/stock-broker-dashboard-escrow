# Stock Broker Dashboard

## Description
A web-based stock broker client dashboard with separate backend and frontend. Users can login, subscribe to stocks, and view live stock price updates. Multi-user support is included.

## Features
- User login via email
- Subscribe to supported stocks: GOOG, TSLA, AMZN, META, NVDA
- Real-time stock updates with Socket.IO
- Multi-user and multi-tab support
- Responsive UI

## Technologies
- Backend: Node.js, Express.js, Socket.IO
- Frontend: React.js, HTML, CSS, JavaScript
